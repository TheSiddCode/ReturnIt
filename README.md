# 🚀 TagBack — Smart Lost & Found System

Empowering people to recover lost items using QR technology.

---

## 📌 Overview

TagBack is a full-stack web application that helps users recover lost items using QR codes.  
Each item is linked to a unique QR code. When someone finds it, they can scan and directly connect with the owner.

---

## ✨ Features

- Secure User Authentication (JWT)
- Add & Manage Lost Items
- QR Code Generation
- Scan QR to Identify Owner
- Chat between Finder & Owner
- Razorpay Payment Integration
- Notification System
- User Dashboard

---

## ⚙️ Setup Instructions

### Clone Repository


git clone https://github.com/TheSiddCode/tagback_clean.git

cd tagback_clean


---

### Backend Setup


cd backend
npm install
npm run dev


Create a `.env` file in backend:


MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
RAZORPAY_KEY=your_key


---

### Frontend Setup


cd frontend
npm install
npm run dev
